Al-Islamiyyah Gunungjati

Pondok Pesantren Al-Islamiyyah Gunungjati berkomitmen untuk membantu keluarga muslim yang mengalami keterbatasan ekonomi agar dapat mengakses pendidikan berkualitas. Dengan tujuan mencetak generasi Muslim hebat.

Program Unggulan Pondok Pesantren Al-Islamiyyah Gunungjati

Program Tahsin Al-Qur'an

Program yang dikhususkan untuk para santri guna memperbaiki bacaan Al Qur’an.

Program Tahfidz Al-Qur'an

Program untuk para santri guna menambah dan menjaga hafalan Al Qur’an Juz 30.

Program Qiroatul Kitab

Programdikhususkan untuk santri guna memperoleh wawasan keagamaan dari sumber otoritatif.
